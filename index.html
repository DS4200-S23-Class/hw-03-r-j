Marks

  • Area (states in the United States)

Channels

  • Color
    • Use color hue to represent attribute of categorical type, red is for Trump, and blue is for Clinton
    • It is a good mapping because colors are very differentiable, and viewer can tell from this visualization immediately which candidate would have been elected by women in each state
  • Size → Area
    • Represents attribute of categorical type, different area represent the state in U.S.
    • This is a good mapping because it shows which states vote for Clinton/Trump and how larger the size corresponding to that color hue indicates that more people vote for that candidate.
  • Position → Horizontal
    • Electoral votes needed to win represents attribute of ordered for position on common scale
    • This is a good mapping because it displays clearly on which candidate received the majority of votes and whether or not they won

Colormap(s)

  • Categorical
    • Represents attribte of categorical type (Clinton/Trump)
    • This is a good mapping because the color choice are roughly the same saturation and value, but perceptually distant. Viewer can easily distinguish from visualization that woman prefer to vote for Clinton rather than Trump in 2016


Marks

  • Points (points on the graph represent each batted ball)
  • Lines (there are horizontal lines in the graph that represent classification of the hit)

Channels

  • Color
    • Based on the scale at the top-left of the visualization, using color saturation to represent attribute of qualitative type
    • This is good mapping because it give more information to the viewers, and they can easily tell how good that hit is in terms of scoring value
  • Position → both
    • represents the exit velocity(x) and launch angle(y) of each batted ball
    • This is a good mapping because viewers can determine what launch angle and exit velocity that player use and how it differs from the other points
  • Position → Horizontal
    • Use scoring value bar to represent attribute of ordered, quantitative for position on a common scale
    • This is a good mapping, because you can see how the color change corresponding to different scoring value

Colormap(s)

  • Divergent
    • Represents attribute of ordered, quantitative
    • divergent color map hue that use white as the midpoint color, purple indicates socring value below average, and orange indicates scoring value above average
    • This is a good mapping because viewers can clearly see if scoring value is below, in the middle, or above the average
